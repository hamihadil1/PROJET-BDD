# Projet Base de Données - Gestion Universitaire

## Description
Système de gestion de base de données pour une université.

## Structure
- **application/**: Code source Python
- **database/**: Fichiers SQL et scripts

## Installation
1. Installer les dépendances: `pip install -r requirements.txt`
2. Importer la base de données
3. Lancer: `python app.py`
